## Installation
### Prerequisites
Ensure you have **Flutter** installed. If not, download it from [flutter.dev](https://flutter.dev/).

### Clone the Repository
```bash
git clone https://github.com/your-username/tuneify.git
cd tuneify
```

### Install Dependencies
```bash
flutter pub get
```

### Run the App
For Android & iOS:
```bash
flutter run
```
For Web:
```bash
flutter run -d chrome
```
